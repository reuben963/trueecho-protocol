# 🌌 TrueEcho Protocol (Core Engine & Medusa Plugin)

> The world's first cryptographically verified, omnichannel, proof-of-transaction review network. Built to eradicate fake reviews, bots, and corporate review manipulation.

## 🚀 The Core Philosophy
Traditional review networks (Google, Yelp, Amazon) are broken. They rely on "open text boxes" easily exploited by click farms, bots, and malicious competitors. 

TrueEcho shifts reviews to an immutable **Proof-of-Transaction** framework. If a user does not possess a cryptographically signed, single-use transaction hash generated by a payment processor at checkout, the ledger physically rejects the submission.

## 🤝 Getting Started
1. Clone the repository.
2. Install dependencies: `npm install`.
3. Link your local test environment variables to a Base Goerli Testnet RPC node.
4. Run the Medusa development server: `medusa develop`.
